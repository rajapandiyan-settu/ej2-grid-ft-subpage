Syncfusion React Grid lets you enable multiple filter modes with filterSettings.type. The Filter Bar adds inline inputs under headers, while Menu and Excel-like filters provide operator dialogs and checkbox-based value selection.

Choose the right approach

Filter Bar

Displays a row of input fields beneath the header. Use Immediate mode for live filtering or OnEnter mode to apply criteria only after pressing Enter.

Checkbox Filter

Renders a checkbox list of unique values so users can select several items at once. Ideal for categorical columns with repeated values.

Excel-like Filter

Opens a dialog with checkbox selection, search, and sorting. Great for complex filtering scenarios and distinct value lists.

Menu Filter

Opens a filter menu dialog from the header icon. Supports operator selection and precise input for each column.

Frequently asked questions

Does filtering support remote data?

Yes. React Grid can send filter queries to your API and render filtered results from remote sources.

Which filter mode is best for dashboards?

Menu filtering works well for dense dashboards because it keeps the table compact while still allowing rich criteria.

Can I use multiple filter modes?

Each grid instance uses one mode at a time, but you can switch modes dynamically for different views or user roles.

Is checkbox filtering good for categories?

Absolutely. Checkbox filter is ideal for repeated values and situations where users need to include several categories.
