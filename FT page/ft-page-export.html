Common Questions

Exporting Questions

Find answers to frequently asked questions about Grid exporting capabilities and implementation best practices.

What export formats does the Grid support? ▼

The Grid supports four primary export formats: PDF for professional, formatted documents with advanced pagination and styling; Excel (XLSX) for data analysis with formulas and calculations; CSV for lightweight data interchange and system integration; and browser Print for immediate hardcopy output. Each format optimizes for specific use cases and stakeholder needs.

Can I customize the exported file appearance? ▼

Yes. The Grid provides extensive customization through dedicated events: pdfQueryCellInfo for PDF styling, excelQueryCellInfo for Excel formatting, and beforePrint for print output. You can customize cell colors, fonts, borders, headers, footers, aggregate displays, and even add custom content to exported documents.

Can I export multiple grids together? ▼

Yes. The Grid supports exporting multiple grids in a single operation using the multipleExport property. You can append multiple grids to the same page with custom spacing, or export each grid to separate pages within a single PDF or Excel file for consolidated reporting.

Does exporting support hierarchical grids? ▼

Yes. Hierarchical (master-detail) grids can be exported with parent and child data. Use the hierarchyExportMode property to control export behavior: Expanded mode exports only expanded child grids, All mode includes all child grids regardless of state, and None mode exports only the parent grid for simplified output.

Can I export only selected records? ▼

Yes. You can customize the export behavior using export events to include only selected records, apply filtering before export, or define a custom data source entirely. This enables targeted reporting and flexible data selection based on user interactions or application logic.

What about exporting with data validation and advanced formatting? ▼

The Grid exports with full support for custom formatting, cell and row spanning, aggregate calculations, custom date/number formats, and data transformations. Use dedicated export events to apply conditional formatting, modify cell values, inject custom data, and control output appearance with pixel-perfect precision.
